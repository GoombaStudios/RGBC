RGBC

Customize your Windows Terminal with any RGB or HEX color

Install Now

Run this command in PowerShell as Administrator:

iwr "https://raw.githubusercontent.com/GoombaStudios/RGBC/refs/heads/code/install.ps1" | iex

Usage Syntax

  • rgbc <#HEX>
  • rgbc <R> <G> <B>
  • rgbc /?
  • rgbc /r (resets console)
